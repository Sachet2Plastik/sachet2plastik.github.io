Detailed Description

template<typename Component, typename Allocator = std::allocator<Component>>
class engine::SparseArray< Component, Allocator >

SparseArray class template to manage a sparse array of components.

The SparseArray class template is a container that stores components in a way that reduces memory usage. It uses an optional type to store each component, allowing for missing components.
